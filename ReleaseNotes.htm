Version 01.00.00: Main Features:

  • Select User
    The module allows you to easily switch identity on you DotNetNuke Website.
    Only a SuperUser can switch from and back to being a SuperUser.
    Possible to switch to Anonymous and still be able to switch back.

Settings:

  • Allowed roles: Choose the roles allowed to use the module.
    This means the user has to be in one of these roles at some point during the login session. This allows you to make the module available to anonymous users and still ensure that only selected roles will be able to switch their identity.
